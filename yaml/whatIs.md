# What is yaml/ yml
- Yaml is an object oriented file. Similar to JSON.
```
Json

{
    servers: [
        {
            name: server1,
            owner: john,
            created: 12/2
            status:active
        }
    ]
}
```
```
Yaml

servers:
    name: server1
    owner: john,
    created: 12/2
    status: active
```
