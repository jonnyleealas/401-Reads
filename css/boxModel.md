# CSS Box Model
- Margin Top
- Border Top
- Padding Top
- Margin Bottom
- Border Bottom
- Padding Bottom
- Margin Left
- Border Left
- Padding Left
- Margin Right
- Border Right
- Padding Right

# Summary
- The css box model is the basics of css layout when looking at the inspect tool in chrome or other browsers.
- Its basically a square with 3 layers --> border, margin, padding, top, bottom, left, right.
